# 🧬 FitGenie

AI-powered personalized nutrition and workout planner.

---

## 🚀 Features

- 🧍 Personalized user profiling based on goals and lifestyle  
- 🍽️ AI-generated full-day meal plans  
- 🏋️ Customized 7-day workout routines  
- 🎥 Demo videos for each workout exercise  
- 📸 Meal scanner to analyze food from uploaded images  
- 🧪 Nutritional breakdown (calories, protein, carbs, fats)  
- 📊 Smart recommendations based on user input  

---

## 🧠 Core Idea

FitGenie uses AI to generate personalized diet and workout plans tailored to each user, making fitness planning more efficient and data-driven.

---

## ⚙️ Tech Stack

- Backend: Flask / Django  
- Frontend: HTML, CSS, JavaScript  
- Database: SQLite / MySQL  
- AI Integration: Gemini API (Google)  
- Image Processing & Nutrition Analysis  

---

## 📸 Highlights

- AI-powered meal and workout generation using Gemini API  
- Integrated workout demo videos for guided training  
- Food image analysis with nutritional insights  
- Clean and intuitive user experience
